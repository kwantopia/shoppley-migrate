Shoppley {% trans "TXT Commands" %}

Just remember the first letter of what you want to do and it's just a very simple txt message to interact with Shoppley!

Merchant

#m email zipcode business_name
#merchant email zipcode business_name
Sign up for a merchant account.
return: A random password is returned that user uses to login to the web site.
example: #merchant joes@bakery.com 02142 Joe’s Bakery
#o offer_description
#offer offer_description
Creates an offer. offer_description is recommended to be within 80 characters.
return: A tracking_code that can be used with #status commands
example: #offer Extra $5 off any pair of pants that are on sale
#s
#status
Checks the status of the latest offer.
return: How many people have been reached and how many redeemed.
#s tracking_code
#status tracking_code
Checks the status of the offer with tracking_code.
return: How many people have been reached and how many redeemed.
example: #status itrwpo
#r offer_code customer_number
#redeem offer_code customer_number
Redeems the customer’s offer. You will receive points back. customer_number is the customer’s phone number. Customer’s phone number is used to verify that the offer belongs to the customer and to reward points to the customer.
return: Confirmation or error that the reward code is invalid
example: #redeem 728234 6172372009
#b
#balance
return: The point balance on your account
#z new_zip
#zip new_zip
#zipcode new_zip
Change the active zip code
return: Confirmation or error that the zip code is invalid
example: #zipcode 94112 (changing to San Francisco for mobile merchant)
#h
#help
return: List of commands and brief descriptions

Customers

#c email zip_code
#signup email zip_code
Sign up new user with email address email and zip_code.
return: A random password is returned that you can use to login to the web site.
example: #signup john@example.com 02139
#i
#info
return: Detailed information about the most recent offer code which details the store location and phone number.
#i offer_code more_offer_codes
#info offer_code more_offer_codes
Offer codes are listed separated by space.
return: Detailed information about one or more offer codes which details the store location and phone number.
example: #info 823145 982415
#f offer_code friend_num more_friend_nums
#forward offer_code friend_num more_friend_nums
Forwards the offer_code to one or more number of friends. List the phone numbers separated by space. When your friends redeem an offer, you will get points.
example: #forward 982415 6179092424 781-234-2342 9782342225
#w any message, brand, product, service describing what you want
#iwant any message, brand, product, service describing what you want
Submit to Shoppley what type offers you would like to receive
example: #iwant offers on cheap designer jeans
#stop
Stop receiving offers
#start
Start receiving offers
#b
#balance
return: The point balance on your account
#z new_zip
#zip new_zip
#zipcode new_zip
Change the active zip code
return: Confirmation or error that the zip code is invalid
example: #zipcode 94112 (changing to receive offers from San Francisco)
#h
#help
return: List of commands and brief descriptions
